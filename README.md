# lovec-wasm

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

WebAssembly (wasm) test in C.

## Features
- WebAssembly (wasm) example in C
- Includes a simple function that adds 1 to the input parameter

## Requirements
- [clang](https://students-tech.blog/post/install-clang.html)
- [Deno](https://deno.com/)

## Usage
1. Edit the [love.c](love.c) file.
2. Run the following commands:
   ```
   sh c.sh
   deno run -A love.js
   ```
3. Or, open the [index.html](https://code4fukui.github.io/lovec-wasm/) file in your browser.
