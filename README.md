# lovec-wasm

WebAssembly(wasm) test in C

## Prepare

- install [clang](https://students-tech.blog/post/install-clang.html)
- install [Deno](https://deno.com/)

## How to run

edit [love.c](love.c)
```
make clean; make
deno run -A love.js
```
or open [index.html](https://code4fukui.github.io/lovec-wasm/) on your browser
